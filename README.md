[AssertJ Extensions](https://github.com/danhaywood/java-assertjext)


/!\ AssertJ version depends on Java version:
- Java6 => AssertJ version 1.x.x
- Java7 => AssertJ version 2.x.x
- Java8 => AssertJ version 3.x.x
